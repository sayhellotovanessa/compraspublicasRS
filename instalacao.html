Compras Públicas RS - Versão 1.0

Aplicativo de visualização de compras do RS

Obrigada por usar nosso aplicativo aberto, ele serve para te ajudar.

O CMPRS (Compras públicas RS) é um aplicativo desenvolvido para participar da primeira edição do Decorders, da W3C. Serve para visualizar e compartilhar os dados de compras e serviços adquiridos pelo governo do Rio Grande do Sul no período de informações deisponibilizadas no site Informações Públicas (www.informacoes.rs.gov.br).

Instalação

  1. Faça download dos arquivos no repositório público /compraspublicasRS no Github.
  2. Descompate o arquivo no seu servidor.
    1. Atenção

      Você precisa rodar os arquivos - ou seja, descompactá-los - em um diretório de arquivos dentro de um servidor web, que tenha suporte para rodar php.

    2. Não precisa instalar nada em banco de dados - o aplicativo consulta arquivos que estão em seu diretório, no formato JSON.
  3. Acesse o diretório para onde você colou os arquivos; o aplicativo deve estar rodando imediatamente.
